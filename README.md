# RawStacker

An app created in Xcode for iOS 16.0 - 26.x and onwards to take RAW images on iOS devices that support capturing them, with the ability to stack them into a TIFF or JPEG.

Things to Work On:

- Rotate Support - Currently the app UI does not have rotation support, meaning the layout breaks when the phone is in landscape, rather than portrait.

- Occasional Crashes on some phones - I am looking into this

**Devices Supported (theoretically):**

- iPhone SE 2nd Gen (tested working)

- iPhone SE 3rd Gen

- iPhone 8/8 Plus

- iPhone X/XS/XS Max/XR

- iPhone 11/11 Pro/11 Pro Max (iPhone 11 tested working)

- iPhone 12/12 Mini/12 Pro/12 Pro Max

- iPhone 13/13 Mini/13 Pro/13 Pro Max

- iPhone 14/14 Plus/14 Pro/14 Pro Max

- iPhone 15/15 Plus/15 Pro/15 Pro Max

- iPhone 16/16e/16 Plus/16 Pro/16 Pro Max

- iPhone 17/17e/17 Pro/17 Pro Max

- iPhone Air
